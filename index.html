RGeo

Support for geo data and databases in ruby and rails

RGeo

  • RGeo: Geospatial data types for Ruby
  • RGeo::ActiveRecord: RGeo integration with ActiveRecord
  • RGeo::GeoJSON: GeoJSON formatting and parsing in Ruby
  • activerecord-postgis-adapter: ActiveRecord adapter for PostGIS
  • activerecord-mysql2spatial-adapter: ActiveRecord adapter for MySQL spatial extensions
  • activerecord-spatialite-adapter: ActiveRecord adapter for Spatialite
  • RGeo::Shapefile: Reader for ESRI Shapefiles in Ruby
